Harvard Square Homeless Shelter

Web App - Volunteer Portal

Project Details

Client: Harvard Square Homeless Shelter

Date: 2018-2019

Visit: Official Website


  • The Harvard Square Homeless Shelter serves the community of individuals experiencing homelessness in Cambridge and Boston. The volunteer network provide guests with shelter, food, security, and a supportive environment of mutual respect.

  • React.js, React Native, Firebase
  • Material-UI, React Browser Router

    • 1. Front-End

    Created the main navigation panel component utilizing Material UI API and set up the front-end UI architecture for the volunteer portal.

    2. Features

    Developed a responsive table component to display and log volunteer actions like client info, supply info, and additional notes. Volunteers are able to track past actions and identify recurring guests.

    3. JumboCode

    Worked with team's mobile, back-end and web teams to develop software that is used by the Harvard Square Homeless Shelter. Project is on-going as part of JumboCode - Tufts University.
